# Getting Started with Steno

This guide will help you set up your first static site using Steno.

## Prerequisites

*   **Deno:** Ensure you have Deno installed. You can download it from [deno.com](https://deno.com/).

## 1. Create a New Project

Start by creating a new directory for your Steno project:

```bash
mkdir my-steno-site
cd my-steno-site
```

## 2. Initialize Steno Configuration

Steno uses a configuration file to define your site's settings. Create a `content/.steno/config.yml` file:

```bash
mkdir -p content/.steno
touch content/.steno/config.yml
```

Open `content/.steno/config.yml` and add the following basic configuration:

```yaml
# content/.steno/config.yml
title: "My Awesome Steno Site"
description: "A static site built with Steno"
author: "Your Name"
contentDir: "content"
output: "dist"
```

## 3. Create Your First Content Page

Steno processes Markdown files. Let's create an `index.md` file in your `content` directory:

```bash
touch content/index.md
```

Open `content/index.md` and add some Markdown content with frontmatter:

```markdown
---
title: "Welcome"
layout: "default" # We'll create this layout next
---

# Welcome to My Steno Site!

This is your first page, generated by Steno.
```

## 4. Set Up a Basic Theme

Steno uses themes to render your Markdown content into HTML. For a quick start, we'll create a minimal theme.

First, create a `theme` directory and a `theme.yaml` file inside it:

```bash
mkdir theme
touch theme/theme.yaml
```

Open `theme/theme.yaml` and add:

```yaml
# theme/theme.yaml
name: "my-basic-theme"
version: "1.0.0"
```

Next, create a `layouts` directory inside your `theme` directory and a `default.scr` file for your layout:

```bash
mkdir theme/layouts
touch theme/layouts/default.scr
```

Open `theme/layouts/default.scr` and add the following Scribe template:

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>{ title } - { site.title }</title>
</head>
<body>
    <header>
        <h1>{ site.title }</h1>
        <nav>
            <a href="/">Home</a>
        </nav>
    </header>
    <main>
        {@html content}
    </main>
    <footer>
        <p>&copy; { new Date().getFullYear() } { site.author }</p>
    </footer>
</body>
</html>
```

Finally, update your `content/.steno/config.yml` to point to your new theme:

```yaml
# content/.steno/config.yml
title: "My Awesome Steno Site"
description: "A static site built with Steno"
author: "Your Name"
contentDir: "content"
output: "dist"
custom:
  theme: "./theme" # Add this line to specify your local theme
```

## 5. Build Your Site

Now you're ready to build your site! You'll need to run Steno with the appropriate Deno permissions.

```bash
deno run -A --unstable jsr:@steno/steno build --config content/.steno/config.yml
```

This command will:
*   Read your `content/index.md` file.
*   Apply the `default.scr` layout from your `theme`.
*   Generate `dist/index.html`.

## 6. View Your Site

Open the generated `dist/index.html` file in your web browser to see your first Steno site!
